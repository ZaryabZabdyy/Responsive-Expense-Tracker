Digital Khata - Expense Tracker API

A simple full-stack JavaScript application built with Node.js, Express, and HTML/CSS to demonstrate backend API development, input handling, data validation, and responses.

Make sure you have " Node.js " installed on your computer.



Validations & Features:

The API strictly validates all incoming user input before saving:

 1- Data Validation: Built-in backend checks to ensure description fields are not empty and 
    expense amounts are valid, positive numbers.
 2- Error Handling: Returns clear HTTP 400 bad request status codes and 
    user-friendly error messages when validation rules fail.



-- How to Run the Project

Follow these simple steps to run the project on a single port:

1) Open the Project: Open the project folder in VS Code
2) Open a Simple Terminal From View On VS code 
3) Write The Following command: node server.js
4) Copy the URL & paste it on any browser 

hint: URL Just Like That ,etc http://localhost:5000
